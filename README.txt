Anthony's Workout Tracker
============================
1. Upload all files to a GitHub repo.
2. Enable GitHub Pages in repo settings (main branch, root folder).
3. Open on your phone and "Add to Home Screen" to install as an app.
4. Tracks workouts offline with PWA support.
5. Exports CSV logs for use in Google Sheets.

Google Sheets Setup:
- Create a Sheet with headers: Date, Day, Exercise, Weight, Sets, Reps.
- Use CSV export (Anthony_workouts.csv) and import manually (or link with Google Apps Script).
